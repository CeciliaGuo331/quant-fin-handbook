# 量化金融速通教程

量化金融基本上可以看作是机器学习与资产定价，因此我们主要介绍金融基础知识、资产定价、机器学习三方面的入门方法，假定你已经有经原、高数、线代、概率统计、python 的基础。

## 金融与应用数学

### 金融基础知识

不会金融基础知识也许也能学会资产定价理论，但仍然推荐先看金融基础知识，以便理解衍生品定价。

1. 金融市场学、公司金融。学会分析宏观经济金融环境和公司基本面。

2. 计量经济学、金融时间序列分析。

### 资产定价

资产，即现金流。定价，即确定一个资产的回报率。资产定价，即给定一个资产的未来现金流信息，确定其回报率。

在学习资产定价的同时，学习随机过程，有利于理解。

1. 债券定价模型
    - 债券二叉树定价模型
    - vasicek 债券定价模型

2. 股票定价模型
    - CAPM
    - APT

3. 衍生品定价模型
    - 期权二叉树定价模型

4. 金融计量模型
    - Fama French
    - Fama MacBeth

### 进阶

如果有志于资产定价学术，请学习凸优化、ODE、PDE、测度论、概率论、随机过程、实变函数、泛函分析、抽象代数。
