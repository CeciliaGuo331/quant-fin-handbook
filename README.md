# 量化金融速通教程

量化金融基本上可以看作是机器学习与资产定价，因此我们主要介绍金融基础知识、资产定价、机器学习三方面的入门方法，假定你已经有经原、高数、线代、概率统计、python 的基础。

---

## 金融与应用数学

### 金融基础知识

不会金融基础知识也许也能学会资产定价理论，但仍然推荐先看金融基础知识，以便理解衍生品定价。以下内容在本科前两年会系统学习，这里给出核心脉络：

1. **金融市场学**。理解金融市场的结构与运作：一级市场与二级市场、交易所与 OTC、做市商机制、订单类型（市价单、限价单、止损单）。了解主要资产类别：股票、债券、外汇、大宗商品、衍生品的基本特征。这部分是理解"在哪交易、交易什么"的基础。

2. **公司金融**。理解公司如何做融资决策（股权 vs 债权）、投资决策（NPV、IRR 等资本预算方法）、股利政策（分红 vs 留存）。核心框架是现金流折现（DCF）——将未来的钱折到现在。这部分教会你分析一家公司"值多少钱"。

3. **宏观经济学**。理解利率、通胀、GDP、货币政策与财政政策如何影响资产价格。在量化中，宏观因子（利率、通胀预期、经济增长）是许多资产定价模型的输入。IS-LM 模型、菲利普斯曲线、泰勒规则是核心工具。

4. **计量经济学**。从数据中推断经济关系的方法论。核心是线性回归及其变体：OLS 估计、假设检验、异方差性与自相关性、工具变量法（IV）、面板数据模型（固定效应/随机效应）。计量经济学直接通往资产定价的实证研究（如 Fama-MacBeth 回归）。

5. **金融时间序列分析**。金融数据的特殊性在于时间维度的依赖结构。核心内容：平稳性与单位根检验、ARIMA 模型、波动率建模（ARCH / GARCH 族）、协整与误差修正模型。这些是金融风险管理和波动率预测的底层工具。

### 资产定价

资产，即现金流。定价，即确定一个资产的回报率。资产定价，即给定一个资产的未来现金流信息，确定其回报率。

在学习资产定价的同时，学习随机过程，有利于理解。

#### 理论基础

- **期望效用理论与风险厌恶**。为什么人们要求风险补偿？从 von Neumann-Morgenstern 期望效用出发，推导风险溢价。绝对风险厌恶（ARA）与相对风险厌恶（RRA）的度量是资产定价的微观基础。

- **随机贴现因子（SDF）框架**。所有资产定价模型都可以写成 $P_t = E_t[M_{t+1} X_{t+1}]$ 的形式，其中 $M_{t+1}$ 是随机贴现因子。这一框架统一了时间折现与风险调整——CAPM、CCAPM、APT 都是 SDF 的特例。

- **无套利定价原理**。在一个无套利市场中，两个未来现金流完全相同的资产，当前价格必须相同。这是衍生品定价的基石：用基础资产和无风险债券复制出衍生品的现金流，则衍生品的价格等于复制组合的成本。

#### 债券定价模型

债券是相对确定的现金流资产，定价的核心是对利率期限结构的建模。

- **债券基础**。到期收益率（YTM）、久期（Duration，衡量价格对利率的敏感度）、凸性（Convexity，久期随利率的变化）。即期利率、远期利率、收益率曲线的关系。收益率曲线形状（水平、上行、下行、倒挂）隐含了市场对未来利率和经济的预期。

- **利率二叉树模型**。将利率建模为随时间分叉的二叉树过程，在每个节点上利率有上升/下降两种可能。从终端现金流回溯折现，得到债券价格。Ho-Lee 模型和 Black-Derman-Toy (BDT) 模型是常见的二叉树枝率模型。

- **Vasicek 模型**。利率的连续时间模型：$dr_t = \kappa(\theta - r_t)dt + \sigma dW_t$（Ornstein-Uhlenbeck 过程）。利率均值回复——远期利率不会无限发散。该模型有闭式解，可以得到零息债券价格的解析表达式。

- **Cox-Ingersoll-Ross (CIR) 模型**。在 Vasicek 基础上将波动项改为 $\sigma\sqrt{r_t}dW_t$，保证了利率始终非负。也是均值回复过程，但分布不再是正态，更适合刻画利率的非负性。

- **Hull-White 模型**。Vasicek 的推广：将长期均值 $\theta$ 设为时变函数 $\theta(t)$，可以精确拟合当前的收益率曲线。实践中广泛应用，尤其适合利率衍生品定价。

#### 股票定价模型

股票是现金流不确定的资产——未来的股息不确定，未来卖出的价格不确定。这导致了更高的风险溢价要求和更复杂的定价。

- **CAPM**。市场组合是最优的风险分散。资产的预期超额收益 = $\beta \times$ 市场超额收益。风险来源只有一个——市场风险（系统性风险）。$\beta$ 衡量资产对市场变动的敏感度。CAPM 是最简洁的资产定价模型，也是因子模型的起点。

- **APT**。放松了 CAPM 的严格假设（不需要均值-方差最优、不需要完整的市场组合）。资产的收益率由多个系统性因子的线性组合驱动：每个因子有一个 $\beta$ 和一个风险价格 $\lambda$。没有指定因子的具体内容——因子可以是 GDP 增速、通胀、利率等宏观变量。

- **CCAPM（基于消费的 CAPM）**。将 CAPM 的微观基础追溯到消费决策：投资者的风险厌恶来自消费波动。资产的风险溢价取决于其收益率与消费增长的协方差。这个模型的简洁性在理论上非常优美，但在实证中面临"股票溢价之谜"（Equity Premium Puzzle）和"无风险利率之谜"（Risk-Free Rate Puzzle）的挑战。

- **股息折现模型（DDM）**。股票价格 = 未来所有股息的现值之和。Gordon 增长模型是最简单的版本：假设股息以恒定速率 $g$ 增长，则 $P = D/(r-g)$。$r$ 是资产的必要回报率。这一框架把"定价"分解为两部分：未来现金流的预测 + 折现率的确定。

#### 衍生品定价模型

衍生品——远期、期货、互换、期权——的价值依赖于基础资产的价格路径。定价的核心思想是"无套利复制"。

- **远期与期货定价**。最基本的衍生品：在到期日以约定价格买/卖资产。通过无套利条件，远期价格 = 现货价格 × (1 + 持有成本 - 持有收益)，这个交易叫做"现金与套利"（Cash-and-Carry）。期货与远期的差异来自逐日盯市（marking-to-market）制度。

- **期权基础**。看涨期权（Call）和看跌期权（Put）的收益函数、内在价值与时间价值、实值/平值/虚值。Put-Call Parity 是无套利的基本关系。期权策略组合：牛市/熊市价差、跨式（Straddle）、宽跨式（Strangle）、蝴蝶价差（Butterfly）等——这些策略的价值在于对波动率和方向的不同观点。

- **期权二叉树定价模型（Cox-Ross-Rubinstein）**。将时间离散化，在每个时间步上股票价格上升/下降一定比例，从到期日的已知收益逐步回溯折现，得到期权的当前价格。二叉树在步长趋于零时收敛于 Black-Scholes。它的优势在于可以处理美式期权（可提前行权）和支付股息的期权。

- **Black-Scholes-Merton 模型**。连续时间下的期权定价公式。核心假设：股票价格服从几何布朗运动、无套利、连续交易。通过 Delta 对冲组合推导出 Black-Scholes PDE：$\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} + rS\frac{\partial V}{\partial S} - rV = 0$。该方程的解给出了欧式期权的闭式价格。

- **Greeks（希腊字母）**。期权价格对各参数的敏感度：Delta（价格对标的资产价格的变化）、Gamma（Delta 的变化率）、Theta（时间衰减）、Vega（对波动率的变化）、Rho（对利率的变化）。Greeks 是期权交易和风险管理的核心工具——一个期权交易者的日常工作就是管理其组合对各 Greeks 的暴露。

- **随机波动率模型**。Black-Scholes 假设波动率恒定，与市场观测（波动率微笑/偏斜）不符。Heston 模型引入波动率自身的随机过程，可以拟合市场中的隐含波动率曲面。局部波动率模型（Dupire）和 SABR 模型也是重要的发展。

#### 因子模型与实证资产定价

资产定价的实证研究试图回答两个核心问题：什么因素决定了不同资产之间的收益率差异（横截面）？什么因素驱动了总体市场回报的时间变化（时间序列）？

- **Fama-French 三因子模型**。在 CAPM 的市场因子之外，加入规模因子（SMB，小盘股 vs 大盘股）和价值因子（HML，高账面市值比 vs 低账面市值比），显著提升了对股票横截面收益的解释力。后续又扩展为五因子模型（加入盈利能力 RMW 和投资 CMA），以及将动量（Momentum）因素加入的六因子版本。

- **Fama-MacBeth 回归**。实证资产定价的核心方法论：两阶段回归。第一阶段：对每只股票在滚动时间窗口上回归，得到其对各因子的 $\beta$ 暴露（时序回归）。第二阶段：在每个截面上，将各股票在该期的收益率对第一阶段估计的 $\beta$ 做回归，估计因子风险溢价 $\lambda$（截面回归）。最终对时间序列上的 $\lambda$ 取平均并做统计推断。这一方法优雅地处理了"$\beta$ 需要估计"和"收益率截面相关"的计量问题。

- **动量与反转效应**。动量（过去 6-12 个月的赢家继续跑赢）和短期/长期反转是两大挑战有效市场假说的异象。动量在几乎所有资产类别和国家市场都存在，是最稳健的因子之一。但动量策略面临尾部风险——动量崩盘（Momentum Crashes）会在市场从极端状态反转时发生巨大的回撤。

- **多因子模型与 Barra 风险模型**。从学术因子到业界实践：Barra 将股票风险分解为行业因子、风格因子（如 Size、Value、Momentum、Volatility、Quality、Growth 等）和特异性风险。多因子风险模型是量化组合管理的核心基础设施。

### 应用数学工具

量化金融中的数学工具远超基础课堂，这里列出最重要的几个方向：

- **随机过程**。这是衍生品定价的数学语言：布朗运动（Brownian Motion）、几何布朗运动（GBM）、伊藤引理（Itô's Lemma）、随机微分方程（SDE）、鞅（Martingale）与等价鞅测度。理解伊藤引理是理解 Black-Scholes 推导的关键——它告诉你一个随机变量的函数如何随时间演化。

- **数值方法**。当闭式解不存在时，如何求解定价方程：蒙特卡洛模拟（模拟大量随机路径，取收益期望的折现——灵活但收敛慢）、有限差分法（在网格上数值求解 PDE——精度高但维数灾难）、二叉树/三叉树（离散逼近——直观，适合美式期权）。

- **最优化**。组合管理中无处不在：均值-方差最优化（Markowitz）及其各种约束变体；凸优化和二次规划是组合配置的基本工具。在机器学习中：梯度下降及其变体（SGD、Adam）是训练模型的基石。拉格朗日乘子法和 KKT 条件是对偶性和约束优化的理论基础。

- **贝叶斯统计**。量化金融中不确定性量化的重要框架：先验分布 → 似然函数 → 后验分布。Markov Chain Monte Carlo（MCMC）是常用的采样推断方法。Black-Litterman 模型就是贝叶斯方法在组合管理中的经典应用——将主观观点与市场均衡权重以定量的方式相结合。

### 进阶

如果有志于资产定价学术，请学习凸优化、ODE、PDE、测度论、概率论（在此基础上学随机过程）、实变函数、泛函分析、抽象代数。

- 连续时间金融需要 SDE 和 PDE 的数学基础，而 SDE 的严谨处理需要测度论概率论（如 Radon-Nikodym 导数用于测度变换、Girsanov 定理用于改变概率测度下的漂移项）。
- 泛函分析在连续时间资产定价的高维推广、期限结构模型的函数型主成分分析中发挥重要作用。
- 抽象代数的一些结构思维也会在理解某些金融数学模型时带来启发。

## 计算机

### 基础知识

- **数据结构与算法**。面向对象编程是构建复杂系统的基础思维。核心数据结构——链表、栈、队列、优先队列（堆）、树、二叉树、图——是高效处理金融数据的底层工具。排序算法、动态规划、DFS / BFS、图相关算法了解即可，但理解其时间复杂度分析（大 O 表示法）对实际工程决策非常有用。

- **Python 科学计算栈**。NumPy（矩阵运算与线性代数）、Pandas（表格数据处理与时间序列操作）、SciPy（科学计算与最优化）、Matplotlib / Seaborn / Plotly（数据可视化）。这是量化金融中最常用的工具链。

- **数据库**。金融数据规模大且需要高频率更新。SQL（关系型数据库）是基本工具，用于查询结构化金融数据（公司报表、价格序列）。了解时间序列数据库（如 InfluxDB、ClickHouse）在存储高频交易数据中的优势。

- **工作环境**。推荐试用 Windows + Linux 双系统，在学习机器学习时方便配置环境。所有流行的量化框架和深度学习库都在 Linux 下获得最佳支持。
  - 中等预算：MacBook 或 Windows 轻薄本
  - 预算较高：Windows 较轻薄的性能本（有 NVIDIA 显卡可用于 CUDA 加速）
  - 预算充裕：MacBook + **非 Mac 的**主机（带 NVIDIA GPU），Mac 用于日常开发，Linux 主机用于模型训练

- **计算机入门**。对于没有系统学过计算机的同学：
  - [Getting Started Handout](https://github.com/ZangXuanyi/getting-started-handout) — 北大民间计算机入门指南，非常推荐
  - [The Missing Semester of Your CS Education](https://missing.csail.mit.edu/) — MIT 的课程，涵盖 Shell、Git、Vim、调试、元编程等实用技能

- **版本控制（Git）**。无论是个人研究还是团队协作，Git 都是现代软件工程的基础设施。至少要学会：clone、add、commit、push、pull、branch、merge、rebase — 以及如何解决合并冲突（Merge Conflict）。

### 机器学习

机器学习是量化金融中挖掘数据模式、构建预测模型的核心工具。大致可以分为监督学习（有标签，如预测收益率方向）、无监督学习（无标签，如市场状态聚类）和强化学习（序贯决策，如最优执行）。

#### 经典机器学习

在进入深度学习之前，经典方法往往更简单、更可解释，在金融领域仍然广泛使用：

- **线性模型与正则化**。线性回归、Ridge（L2 正则化）、Lasso（L1 正则化，可做变量选择）、Elastic Net。对于金融数据常见的高噪声、小样本场景，正则化是防止过拟合的关键。

- **树模型与集成方法**。决策树 → 随机森林（Bagging）→ 梯度提升树（GBDT: XGBoost / LightGBM / CatBoost）。树模型天然处理缺失值和特征间的非线性交互，在量化选股中非常流行。XGBoost 和 LightGBM 是 Kaggle 竞赛中的常胜将军，也是业界量化因子挖掘的主力工具。

- **支持向量机（SVM）**。通过核技巧（Kernel Trick）将数据映射到高维空间进行线性分类/回归。在小样本但高维度的金融场景中有一定优势。

- **聚类与降维**。K-Means、层次聚类用于市场状态的划分（牛/熊/震荡）；PCA 用于因子降维、去噪和构建统计因子模型；t-SNE / UMAP 用于高维数据的可视化探索。

#### 深度学习

深度学习在处理非结构化数据（如图表、文本、订单流）和捕捉复杂非线性模式上有显著优势：

- **深度学习基础**。
  - 前馈神经网络（MLP）—— 将输入变量非线性变换为输出预测，是深度学习的原子构件。
  - [CS229: Machine Learning](http://cs229.stanford.edu/)（Stanford，Andrew Ng）—— 经典机器学习课程，数学推导严谨，涵盖从线性回归到 SVM 到深度学习的核心算法。
  - [Dive into Deep Learning](https://d2l.ai/) —— 交互式在线教材，从零实现到现代架构，代码与理论结合，适合动手学习。

- **计算机视觉**。[CS231n: Convolutional Neural Networks for Visual Recognition](http://cs231n.stanford.edu/)（Stanford）。CNN 在金融中的一个有趣应用是从卫星图像（停车场车辆数量、油罐阴影、农田长势）中提取另类数据信号。

- **序列建模与时间序列**。金融数据天然是时间序列。RNN → LSTM / GRU 处理序列依赖；Temporal Convolutional Networks（TCN）是 CNN 在时间维度上的应用；Transformer 及 Attention 机制是当前序列建模的最前沿。时间序列预测的特定挑战包括：非平稳性、低信噪比、结构突变（Regime Change）。

- **自然语言处理（NLP）与金融文本**。从财报、新闻、社交媒体、央行公告中提取信号。词嵌入（Word2Vec、GloVe）→ 上下文嵌入（BERT、FinBERT——金融领域微调的 BERT）→ 大语言模型（LLM）。情感分析是最直接的应用，但更深入的应用包括事件抽取（并购公告、诉讼）和关系抽取（供应链依赖）。[CS336: LLM from Scratch](https://cs336.stanford.edu/) 是从底层理解 LLM 的好课程。

- **图神经网络（GNN）**。金融市场天然是图结构：供应链关系、行业关联、股东网络、交易对手风险传导。GNN 可以建模公司间的关系交互——例如公司 A 的股价不仅取决于自身的基本面，也受其上下游公司的影响。

- **生成模型**。VAE、GAN、扩散模型（Diffusion Models）用于金融时间序列的生成与增强——在小样本或隐私受限的场景下生成合成数据，或在风险管理中进行情景生成（Scenario Generation）。

#### 强化学习

[CS285: Deep Reinforcement Learning](http://rail.eecs.berkeley.edu/deeprlcourse/)（UC Berkeley）。

强化学习处理序贯决策问题——在每一个时间步选择行动以最大化累积回报。在量化金融中的应用包括：

- **最优执行**（Optimal Execution）。将大订单拆分为小订单逐步执行，在"市场冲击（太过激进）"和"价格风险（太过缓慢）"之间权衡。Almgren-Chriss 框架是经典方法，强化学习可以在更真实的市场微观结构下学习执行策略。

- **做市策略**（Market Making）。做市商同时报出买卖价格，通过赚取买卖价差获利，同时管理库存风险。Avellaneda-Stoikov 模型建立了库存风险控制的理论框架，而 RL 可以学习更复杂动态市场环境中的报价策略。

- **动态组合管理**。在连续时间上根据市场状态动态调整资产配置权重，这本质上是一个序贯决策问题。RL 可以学习对多资产组合的动态再平衡策略，可能在交易成本、市场摩擦等现实约束下超越静态方法和 MPC（模型预测控制）。

#### 模型评估与回测

这是量化金融机器学习中最容易出问题、但却最关键的一环：

- **避免前视偏差**。在任何时间点，模型只能使用该时刻之前的信息。在构建特征时，必须确保财务报表的发布日期、价格数据的时区等不产生未来信息泄露。

- **交叉验证的时间序列化**。标准的随机 K 折交叉验证破坏了时间依赖结构，在金融数据中会产生严重的过拟合。使用时间序列交叉验证（滚动窗口/扩展窗口）才符合金融实际——你永远只用过去的数据训练模型，在未来的数据上评估。

- **样本内 vs 样本外**。在样本内拟合得好不等于能预测未来。金融中的信噪比极低，几乎所有复杂的模型在样本外都会大幅衰减。使用样本外（Out-of-Sample）$R^2$ 而不是样本内 $R^2$ 来衡量模型的预测能力。

- **回测（Backtesting）**。将交易策略应用于历史数据，模拟其历史表现。回测中的常见陷阱：幸存者偏差（当前存在的公司回看历史会高估表现）、交易成本与滑点（实际成交价与回测使用价格的差异）、过拟合（尝试 100 个策略，报告最好的那个）→ 需要做多重检验校正。

- **策略的稳健性检验**。一个策略如果在参数微调、时间窗口微调、资产范围微调后都保持稳定表现，才可能不是假阳性。Deflated Sharpe Ratio 和 False Discovery Rate（FDR）控制在量化金融中非常重要。

### 进阶

- **《深入理解计算机系统》（CSAPP）**。理解计算机系统的底层工作原理——内存层次、缓存、虚拟内存、编译链接——对于优化大规模计算任务（例如 GPU 上的蒙特卡洛模拟、高频数据处理中的延迟优化）非常关键。

- **并行与异构计算**。加速金融计算的基本手段。
  - **CUDA**：NVIDIA GPU 的并行编程框架。Monte Carlo 模拟（独立路径天然并行）、神经网络训练、大规模矩阵运算都可以利用 GPU 获得数量级加速。
  - **OpenCL**：跨平台的并行计算框架，不限于 NVIDIA。
  - **OpenGL / Vulkan**：图形 API，在某些高频交易系统中用于渲染可视化。

- **C++ 在量化金融中的应用**。大多数对延迟敏感的交易系统和定价引擎使用 C++。理解内存管理、模板编程、STL 以及 Boost 库（尤其是 Boost.Math 中的统计分布和特殊函数）对于参与 sell-side 量化开发非常重要。

- **分布式计算**。当单机处理能力不够时：Spark（批处理大数据）、Dask（Python 原生的并行计算）、Ray（分布式 ML 训练与推理）。金融中的大规模回测（在 5000+ 只股票 × 20 年日频数据上测试策略）通常需要分布式框架。

## 量化金融前沿与专题

### 市场微观结构

研究订单如何在市场中成交——Tick 数据（逐笔数据）、订单簿动态、价差成分（信息不对称成本 + 存货成本 + 订单处理成本）、市场影响模型（如 Almgren-Chriss）。理解市场微观结构是设计执行算法（尽可能便宜、尽可能隐蔽地完成交易）的基础。高频交易（HFT）是这一领域的最极端应用。

### 另类数据

传统量化信号主要来自价格（技术面）和财报（基本面）。另类数据则来自于非传统来源：
- **文本数据**：新闻、财报电话会议（Earnings Call Transcripts）、SEC 文件、社交媒体
- **消费数据**：信用卡交易、电商销售估计（如 Earnest / Second Measure）
- **卫星数据**：零售停车场、油罐、农作物监测
- **物联网数据**：货船 AIS 信号（供应链跟踪）、航班数据
- **网络抓取数据**：招聘信息（公司扩张信号）、价格爬虫、App 下载排名

另类数据的挑战在于：数据清洗（常常非常 messy）、覆盖范围（可能仅覆盖特定子集）、历史长度（很多来源没有足够长的历史用于回测）、以及法律/合规风险。

### 风险管理

风险管理的核心是理解极端事件——不是那些大概率发生的小损失，而是小概率但一旦发生就会毁灭组合的事件。

- **VaR（Value at Risk）与 Expected Shortfall（CVaR）**。VaR 回答了"在给定置信水平下，最大损失是多少"，但 VaR 不满足次可加性（两个资产组合的 VaR 可能大于各自 VaR 之和），因此 Basel III 用 Expected Shortfall 替代了 VaR。

- **极值理论（EVT）**。传统的正态分布假设严重低估了尾部风险。EVT 专门建模极端事件：分块最大值（Block Maxima）方法和 POT（Peaks Over Threshold）方法。金融收益率的尾部比正态分布厚得多——这就是为什么 2008 年在正态模型下应该是"千年一遇"的事件实际每隔几十年就会发生一次。

- **Copula**。将多个资产的风险依赖结构从各自的边缘分布中分离出来建模。各类 Copula（Gaussian、t、Clayton、Gumbel）刻画不同类型的尾部依赖——例如 t-Copula 有双侧尾部依赖，Clayton Copula 只有下尾依赖（熊市中资产的相关性比牛市中高）。

- **压力测试与情景分析**。给定一个极端场景（如利率上升 200bp、股市下跌 30%、人民币贬值 10%），计算组合的 P&L 变化。压力测试的关键挑战在于定义"可能但罕见"的合理场景。

### 资产配置与组合理论前沿

传统 Markowitz 均值-方差框架对输入参数（期望收益率和协方差矩阵）极为敏感——小的估计误差导致极其不同的组合权重，往往给出不合理的高度集中配置。

- **收缩估计（Shrinkage）**。样本协方差矩阵是最简单的估计但含有大量估计噪声。Ledoit-Wolf 收缩方法将样本协方差向结构化的目标矩阵（如常数相关矩阵或单因子模型协方差）收缩，大幅提升样本外表现。

- **Black-Litterman 模型**。将投资者的主观观点（如"科技股跑赢能源股 5%"）以贝叶斯的方式与市场均衡权重（隐含于 CAPM 反推）融合。结果是在中性配置的基础上，向有信心的观点方向倾斜，且倾斜程度与观点的置信度成比例。

- **风险平价（Risk Parity）**。不是按资金权重来配置，而是按风险贡献来配置——使每个资产对组合总风险的边际贡献相等。桥水基金的"全天候"策略是风险平价的著名实践。在通胀/通缩/增长上行/增长下行四象限中平衡配置，以期在任意经济环境下都不会大幅亏损。

- **分层风险平价（HRP）**。Marcos López de Prado 提出的方法，利用层次聚类替代协方差矩阵求逆，避免了 Markowitz 在协方差矩阵病态时的不稳定问题。在实践中，HRP 通常比变种 Markowitz 和传统风险平价产生更好的样本外表现。

### 行为金融学

经典金融假设人是理性的，市场是有效的。行为金融学将这些假设与现实对照，研究认知偏误如何导致市场失效。

- **前景理论（Prospect Theory）**。Kahneman & Tversky（1979）：人们在盈利区域是风险厌恶的（锁定利润），在亏损区域是风险追逐的（不愿意止损）。损失厌恶——相同金额的损失带来的痛苦约是盈利快感的两倍。这些偏误导致了"处置效应"——过早卖出盈利股票、过久持有亏损股票。

- **过度反应与反应不足**。投资者对一连串好消息反应不足（导致动量效应），而对极端消息过度反应（导致长期反转效应）。这两种偏误同时存在，共同解释了动量与反转这两个表面上矛盾的市场异象。

- **套利限制**。即使理性交易者知道价格是错误的，纠正错误需要：资金（如果错误进一步扩大，做空的保证金会被耗尽）、时间（错误可能持续很长时间）、和机制（不是每种错误都可以被低成本地做空）。套利限制是行为金融解释非有效市场现象的核心机制——噪音交易者风险（Noise Trader Risk）。

### 加密货币与 DeFi 量化

对英国金融背景的学生而言，这是一个快速发展的领域，伦敦也是全球加密货币交易的重要中心。

- **链上数据分析**。区块链的独特优势是数据的高度透明性——每一笔交易都被记录在公开账本上，为量化分析提供了前所未有的粒度。

- **DeFi 协议中的套利**。自动化做市商（AMM，如 Uniswap）的恒定乘积公式 $x \cdot y = k$ 在每次交易后产生与外部市场价格之间的价格差，为套利者创造了机会。闪电贷（Flash Loan）使得零本金套利成为可能。

- **永续合约资金费率**。永续合约（Perpetual）通过资金费率机制锚定现货价格——当合约价格高于现货时，多头向空头支付资金费。资金费率是市场情绪的有用量化信号。

### 学术资源与研究方向

#### 重要期刊
- **Journal of Finance (JF)** — 金融学最高期刊，覆盖面广
- **Journal of Financial Economics (JFE)** — 侧重微观金融与公司金融
- **Review of Financial Studies (RFS)** — 侧重理论与实证
- **Journal of Portfolio Management (JPM)** — 业界导向的组合管理研究
- **Journal of Financial Data Science** — 金融机器学习与数据科学的新兴期刊
- **Mathematical Finance** — 数理金融的数学理论

#### 核心教材
- *Asset Pricing* — John H. Cochrane。资产定价的经典教材，以 SDF 框架统一全书，强调经济学直觉而非纯数学推导。难度适宜（需要一定的计量和随机过程基础）。
- *Stochastic Calculus for Finance* (Vol I, II) — Steven E. Shreve。金融随机分析的标准教材：Vol I 是离散时间模型（二叉树），Vol II 是连续时间模型（布朗运动、伊藤积分、Black-Scholes、利率模型）。
- *Options, Futures, and Other Derivatives* — John C. Hull。衍生品定价的标准参考书，覆盖面广，适合自学。
- *Advances in Financial Machine Learning* — Marcos López de Prado。将机器学习应用于量化金融的实践指南，涵盖了数据标签方法、回测过拟合的统计检验、微观结构特征工程等关键实操问题。
- *Active Portfolio Management* — Grinold & Kahn。量化组合管理的"圣经"，引出了信息率（IR）、信息系数（IC）、广度（Breadth）以及基本法则（Fundamental Law of Active Management）：$IR = IC \cdot \sqrt{Breadth}$。
- *The Econometrics of Financial Markets* — Campbell, Lo & MacKinlay。将计量方法系统地应用于金融市场实证研究，涵盖了随机游走检验、事件研究法、CAPM 检验、利率期限结构、非线性模型等核心主题。

#### 重要论文（入门建议阅读）
- Markowitz (1952): "Portfolio Selection" — 均值-方差框架的起源
- Sharpe (1964): "Capital Asset Prices" — CAPM 的诞生
- Fama & French (1993): "Common Risk Factors in the Returns on Stocks and Bonds" — 三因子模型的奠基之作
- Jegadeesh & Titman (1993): "Returns to Buying Winners and Selling Losers" — 动量效应的经典发现
- Carhart (1997): "On Persistence in Mutual Fund Performance" — 四因子模型（加入动量）
- Fama & French (2015): "A Five-Factor Asset Pricing Model" — 五因子模型
- López de Prado (various years) — 金融 ML 的系列工作，包括回测过拟合、元标签、三重障碍法等

## 学习路径

以下按内容模块列出核心学习路径，每个模块内部有先后顺序，但模块之间不一定有严格的依赖关系。你可以并行学习，也可以按自己的兴趣穿插。没有时间线——这不是一份课程表，而是一张地图。

### 路径一：金融理论主线
金融基础知识（金融市场学 → 公司金融 → 宏观经济学）→ 计量经济学 → 资产定价（CAPM → APT → 因子模型 → 衍生品定价）→ 随机过程 → 进阶数学

### 路径二：计算与工程主线
Python 科学计算栈 → 传统算法与数据结构 → 工作环境 → Git → 并行计算 / C++ / CSAPP（进阶）

### 路径三：机器学习主线
经典机器学习（线性模型 → 树模型 → SVM → 聚类/降维）→ 深度学习（MLP → CNN → RNN/LSTM → Transformer → NLP → 生成模型 / GNN）→ 强化学习 → 模型评估与回测

### 路径四：实践与专题主线
因子模型与实证资产定价 → 组合优化 → 另类数据 → 回测方法论 → 风险管理 → 市场微观结构 → 行为金融学

### 主线之间的交汇点
- **资产定价 + 计量经济学** → 因子模型实证研究（Fama-MacBeth 回归、因子构建与检验）
- **资产定价 + 随机过程 + 数值方法** → 衍生品定价与风险
- **因子模型 + 机器学习** → 用 ML 方法挖掘新因子、非线性多因子模型
- **组合优化 + 机器学习** → 动态资产配置、RL 交易策略
- **另类数据 + NLP** → 从新闻/财报/社交媒体中提取投资信号
- **市场微观结构 + 强化学习** → 最优执行算法、做市策略

---

**最后**：量化金融是一个跨度很大、持续演进的领域。这份学习地图的作用不是告诉你"应该去哪儿"，而是让你知道"这里有什么"——看到全貌之后，选你最感兴趣的方向深入。保持好奇心，多动手做实证（从获取数据、清洗、建模到回测的完整流程远胜于只读论文），享受将数学、代码与金融直觉编织在一起的乐趣。
